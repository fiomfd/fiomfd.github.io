Using open data on the web: COVID-19

We draw graphs of time-series data of COVID-19 of cases and deaths by using data sourced by
  • Johns Hopkins Corona Virus Resource Center,
  • Japanese Ministry of Health.
In addition if time-series data of recovered cases is available, one can compute numerical solutions to the initial value problem for the SIR model and similar models.
